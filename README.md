# 🚀 RoleLedger

RoleLedger is a full-stack Role & Permission Management System built using Spring Boot, Angular, and MySQL.  
It implements secure authentication and role-based access control (RBAC).

---

## 🛠 Tech Stack

### Backend
- Spring Boot
- Spring Security
- JWT Authentication
- MySQL

### Frontend
- Angular
- TypeScript
- Bootstrap / CSS

### Database
- MySQL

### Deployment Ready
- Render / Railway (Backend)
- Vercel (Frontend)

---

## ✨ Features

- User Registration & Login
- JWT-based Authentication
- Role-Based Access Control
- Secure REST APIs
- Clean UI with Angular
- Production-ready folder structure

RoleLedger

## ⚙️ How To Run Locally

### Backend
1. Configure MySQL database
2. Update `application.properties`
3. Run Spring Boot application

### Frontend
```
cd frontend
npm install
ng serve
```
